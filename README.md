## ⚡️ Clone the repo

* Clone the repository:
    ```gitbash
    git clone git@github.com:ElmiriYounes/portfolio.git
    ```

* Install the packages (from the root folder):
    ```gitbash
    npm install
    ```

* Get started (from the root folder):
    ```gitbash
    npm start
    ```
